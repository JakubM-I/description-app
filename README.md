# Product Description Generator

Szkielet aplikacji webowej do generowania kodu HTML opisu produktu oraz prostego podglądu tekstowego.

## Etapy

- Etap 0: setup projektu, struktury katalogów i placeholderów
- Etap 1: mock danych i podstawowy flow UI
- Etap 2: Netlify Functions + Airtable Web API
